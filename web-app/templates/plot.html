What does it represents to ?
 The Above Chart represents the Google Stocks for the time-period of March-2021 to July-2021

What tools/technologies are used here ?
 I've used a set of tools/technologies in creating this Web-Graph includes,

  • Technologies - Python
  • Libraries - Pandas, Bokeh, and Flask
  • Development - PyCharm, Jupyter Notebook, Heroku

What are the Features ?
 This is a CandleStick Graph developed using Bokeh (library for Plotting Graphs), this is an interactive graph that can be used for :

  1. Pan - This is the default selected option, so that you can move the graph using the Mouse.
  2. Box Zoom - This option can be used for Zooming-In the graph, in order to view the stats more clearly. this can be done using Mouse and drawing a square on the graph, so it will be in a Zoomed state.
  3. Wheel Zoom - This option can be used to scroll Zoom-in-out through the Mouse's Wheel.
  4. Save - This option can be used to save the following graph into your system in (.png) format.
  5. Reset - This option can be used to reset the Graph, if you've been stucked in Zooming-in or out.

the function came in, for checking the increase and decrease in Stocks for comparing between `Closing price` to `Opening price` and then returning the value, following this, we've reached to the adding columns in our Data frame, here now, i've added three columns to the dataframe (i.e: Status, Middle, and Height), these three are required for the formation of the square in the graph, these as : Status for top, and bottom of the square, Middle for mean of top and bottom values, and Height for the left and right side of the square,
